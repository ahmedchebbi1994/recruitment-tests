# My Books - 1.0 (1)

## Available features

### -----------Display books list
### -----------Display defails of book
### -----------Manage my local basket (add, remove) book
### -----------Display commercial offer depend books added in local basket



###  Language : Swift 5
###  Design Pattern : MVVM-C (Model View ViewModel Coordinator)
###  **Dark** & **Light** Mode available
